## 🎯 Description

- Déployer des skills custom pour besoins métiers

## 🛠️ Stack Technique

```
OpenClaw Core:
├── Gateway (port 18789)
├── Skills (25+ installés)
├── MCP Servers (custom)
└── Sessions (isolated/main)

Integrations:
├── Slack API (notifications, commands)
├── Notion API (knowledge base)
├── GitHub API (code review, issues)
├── WhatsApp Business (alerts)
└── Custom APIs (client-specific)

Deployment:
├── Docker (containers)
├── VPS client ou Toolbox Hub
└── Cloudflare Tunnel (accès sécurisé)
```
